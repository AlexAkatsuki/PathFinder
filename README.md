# Path Finder - Поиск пути на сетке

Приложение для поиска кратчайшего пути на сетке с препятствиями с использованием алгоритма BFS (поиск в ширину).


## Возможности

- Генерация случайной сетки с препятствиями
- Поиск пути алгоритмом BFS (поиск в ширину)
- Установка стартовой и конечной точек
- Предпросмотр пути при наведении курсора
- Масштабирование колесом мыши
- Многопоточные вычисления
- Сохранение положения окна

## Технологии

- **C++17** - основной язык программирования
- **Qt 6.10** - графический интерфейс
- **CMake** - система сборки
- **BFS алгоритм** - поиск кратчайшего пути

## Установка и сборка

### Требования
- Qt 6.10 или выше
- Компилятор с поддержкой C++17
- CMake 3.16+

## Клонирование репозитория
git clone https://github.com/ваш-username/PathFinder.git
cd PathFinder

## Создание папки сборки
mkdir build
cd build

## Конфигурация проекта
cmake ..

## Сборка
make -j4

## Запуск приложения
./PathFinder

## Использование

1. Установите размер сетки
2. Сгенерируйте сетку
3. Установить точки:
    ЛКМ - стартовая точка,
    ЛКМ+shift - конечная точка
4. Найдите путь
5. Предпросмотр - наведите курсор для отображения возможного пути

## Hot Keys

- Ctrl + Колесо мыши - масштабирование
- ЛКМ - установка начальной точки
- ЛКМ + shift - установка конечной точки
